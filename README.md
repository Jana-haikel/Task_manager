# Task_manager
A modern, full-stack task management web application built with vanilla JavaScript, jQuery, PHP, and SQLite. Features a clean UI with calendar view, quick todos, and deadline tracking.

# Project Structure

TASK-MANAGER/
├── api/
│   └── api.php          # RESTful API endpoints
├── css/
│   └── styles.css       # Application styles
├── data/
│   ├── database.db      # SQLite database (auto-generated)
│   ├── tasks.json       # Tasks JSON backup
│   └── todos.json       # Todos JSON backup
├── js/
│   └── app.js           # Frontend JavaScript logic
├── index.html           # Main HTML file
└── README.md

# Tech Stack
 ## Frontend
HTML5 semantic markup
CSS3 with Flexbox/Grid layouts
Vanilla JavaScript (ES6+)
jQuery 3.7.1 for AJAX and DOM manipulation
 ## Backend
PHP 8.x
SQLite database
RESTful API architecture
Dual storage with JSON synchronization

# Prerequisites
PHP 7.4+ with SQLite extension enabled
Web server (Apache, Nginx, or PHP built-in server)
